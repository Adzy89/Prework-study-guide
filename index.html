HTML

  • Hyper Text Markup Language
  • Html tells the website what information to display.
  • There are body and elements in html files.
  • Html is a coded language to display a website.
  • There are 3 languages that make up web development. Html,CSS & JavaScript.
  • Html files contain elements, these elements tell the computer what to display on the page.
  • There are 4 elements, Text, Image, Header and List.
  • As a markup programing language, HTML is used to create static websites.

CSS

  • We use css to style our html pages
  • This makes a website more vibrant and colourful
  • We always create a seperate CSS file for our HTML pages, which makes it easier to make changes if we need to.
  • A CSS rule consists of a selector and a declaration block.
  • CSS can be written directly into the html document. this is called inline CSS.
  • The CSS syntax structure is pretty simple. It has a selector and a declaration block.
  • Internal style sheet contain css rules for the webpage which only appy to that page.
  • when using an external style sheet we sue the link element in our code.

Git

  • GitHub is an online software development platform used for storing tracking, and collaborating on software projects.
  • You can create multipy branches using github, which keeps the orignal code on the main branch.
  • Git is open-source version control software.
  • Version control systems are nessacry for building software because it allows users to make copies of the code without breaking the exisiting soruce code.
  • Users can make pull-requests to merge the code with your approval.
  • Github allows you to create repositaries for which you can store your projects.
  • Git and Github are not the same, however you cannot use Github without Git.
  • Once we have finihsed our work we use the command "commit -m" to add a message to our work, so others users can see what was done.

JavaScript

  • JavaScript is a programming language that adds interactivity to your website.
  • Javascript can create interactive web pages, create mobile applications, create browser extensions, create back-end for servers.
  • A function must be call by name in order to execute.
  • JavaScript has great functionality you can design web-based games, complex applications dynamically creating HTML and setting CSS styles; or generating 3D graphics and audio samples.
  • Javascript has only three data types: string, Number and Boolean.
  • Arrays are used to store multiple values in a single variable.
